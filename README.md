# RAT MP

Cross-platform music player using ImGui inspired by foobar2000. Supported formats: `wav, flac, ogg, opus, m4a, mp3, ape, aiff, aac, alac, wma`.

## Some features
- Powerful custom theming.
- ReplayGain support with optional pre-amp gain.
- Spectrum with multiple modes and oscilloscope visualizers.
- Seekable wave bar.
- Automatic multi-artist/genre tags separated by commas.
- "Radio" mode which automatically builds a list of similar tracks from a given track.
- View library by folders, artists, genres or albums.
